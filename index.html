1. Identify the oxidizing agent in the reaction: Zn + Cu²⁺ → Zn²⁺ + Cu

Solution: Cu²⁺ is reduced by gaining electrons, so it is the oxidizing agent.

2. In the reaction 2Fe³⁺ + Sn²⁺ → 2Fe²⁺ + Sn⁴⁺, which species is reduced?

Solution: Fe³⁺ is reduced to Fe²⁺ by gaining electrons.

3. What is the oxidation number of sulfur in SO₄²⁻?

Solution: Sulfur has an oxidation state of +6 in sulfate ion.

4. Which element is oxidized in the reaction: 2Na + Cl₂ → 2NaCl?

Solution: Sodium (Na) loses electrons, so it is oxidized.

5. In the redox reaction, electrons are:

Solution: Electrons are lost by the reducing agent during oxidation.
